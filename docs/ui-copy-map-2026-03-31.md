# 产品界面文案映射

本次整理聚焦前端用户界面，统一把偏包装感或偏工程感的表达改为更清晰的桌面工具文案。

## 首页与导航

- `新任务台 · NEW MISSION` -> `新任务入口`
- `开启新视角。` -> `开始整理。`
- `新建任务` -> `开始整理`
- `分析文件夹并生成 Windows 图标` -> `选择目标文件夹并生成图标`

## 工作区

- `确认并推送到计划阶段` -> `确认这些归类`
- `AI 引擎扫描中` -> `正在扫描`
- `Final Confirmation | 最终确认` -> `最后确认`
- `当前只能查看之前的方案 and 记录` -> `当前只能查看之前的方案和记录`

## 历史页

- `归档索引` -> `记录列表`
- `会话归档` -> `任务记录`
- `执行归档` -> `执行记录`
- `这次整理具体改了什么` -> `本次变更明细`

## 设置页

- `统一配置内核` -> `模型与工具配置`
- `统一设置收口` -> `模型与工具设置`
- `设置已原子保存，四类配置已一起生效` -> `设置已保存并生效`

- `批量应用` -> `批量应用图标`
